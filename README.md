# Deepika Portfolio (Next.js + TypeScript + Tailwind + Motion)

## 1) Run locally
```bash
npm install
npm run dev
```
Open http://localhost:3000

## 2) Customize
Edit `src/data/portfolio.ts` (name, links, projects, skills).

## 3) Deploy to Vercel
- Push to GitHub
- Import repo in Vercel
- Framework: Next.js (auto-detected)
- Build: `npm run build`
- Output: Next.js default

## 4) Add your resume + photo
Place:
- `public/Deepika_Resume.pdf`
- `public/profile.jpg`
Then update the links in `src/data/portfolio.ts`.
